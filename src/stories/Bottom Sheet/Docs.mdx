## Bottom Sheet List

Placing `zeta-list-item`'s inside `zeta-bottom-sheet` will create a list of items in the bottom sheet.

<Canvas of={BottomSheet.BottomSheetList} />

## Bottom Sheet Grid

Placing `zeta-grid-menu-item`'s inside `zeta-bottom-sheet` will create a grid of items in the bottom sheet.

<Canvas of={BottomSheet.BottomSheetGrid} />

## Bottom Sheet Generic Content

Placing any content inside `zeta-bottom-sheet` will be directly rendered to the bottom sheet.
