Shimmer is a loading skeleton component that displays an animated shimmer effect to indicate content is loading. It provides a smooth, animated placeholder that helps maintain visual continuity while data is being fetched.

## Basic Usage

Use the Shimmer component to show loading states in your application. It automatically adapts to light and dark color schemes.

```html
<zeta-shimmer></zeta-shimmer>
```

## Customization

You can customize the shimmer's dimensions using CSS custom properties:

```html
<!-- Custom height -->
<zeta-shimmer style="--shimmer-height: 30px;"></zeta-shimmer>

<!-- Custom width -->
<zeta-shimmer style="--shimmer-width: 200px;"></zeta-shimmer>

<!-- Both custom height and width -->
<zeta-shimmer style="--shimmer-height: 40px; --shimmer-width: 300px;"></zeta-shimmer>

<!-- Custom animation speed -->
<zeta-shimmer style="--shimmer-animation-duration: 2s;"></zeta-shimmer>

<!-- Rounded corners -->
<zeta-shimmer style="--shimmer-border-radius: 8px;"></zeta-shimmer>
```

## Common Use Cases

**Table Loading States:**

```html
<div style="display: flex; flex-direction: column; gap: 8px;">
  <zeta-shimmer></zeta-shimmer>
  <zeta-shimmer></zeta-shimmer>
  <zeta-shimmer></zeta-shimmer>
</div>
```

**Flex Layout:**

```html
<div style="display: flex; gap: 10px;">
  <zeta-shimmer style="flex: 1;"></zeta-shimmer>
  <zeta-shimmer style="flex: 2;"></zeta-shimmer>
  <zeta-shimmer style="flex: 1;"></zeta-shimmer>
</div>
```

## Accessibility

The Shimmer component is designed to be accessible and provides visual feedback for loading states. It uses semantic HTML and follows WCAG guidelines.
